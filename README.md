# PeaversNeedThat

A World of Warcraft addon that monitors loot drops in M+ dungeons and provides quick whisper dialogs to ask for items.

**Website:** [peavers.io](https://peavers.io) | **Addon Backup:** [vault.peavers.io](https://vault.peavers.io) | **Issues:** [GitHub](https://github.com/peavers-warcraft/PeaversNeedThat/issues)

## Features

- Monitors loot drops in Mythic+ dungeons
- Detects items that are upgrades for your character
- Provides quick whisper dialogs to politely ask group members for items they may not need
- Non-intrusive notifications for loot events

## Installation

1. Download from [CurseForge](https://www.curseforge.com/wow/addons/peaversneedthat)
2. Ensure [PeaversCommons](https://www.curseforge.com/wow/addons/peaverscommons) is also installed
3. Ensure [PeaversConfig](https://www.curseforge.com/wow/addons/peaversconfig) is also installed
4. Enable the addon on the character selection screen

## Usage

- Automatically activates in Mythic+ dungeons
- When loot drops, a dialog appears if an upgrade is detected
- Click to send a polite whisper requesting the item

## Configuration

Access settings through the game's addon options to customize loot tracking and notification preferences.

## Dependencies

- [PeaversCommons](https://www.curseforge.com/wow/addons/peaverscommons) (required)
- [PeaversConfig](https://www.curseforge.com/wow/addons/peaversconfig) (required)
